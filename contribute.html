Contribute to UISpell

Help shape the future of spelling education. Whether you're a developer, educator, or supporter, there are many ways to get involved!

How to Commit to the Repository

1

Fork and Clone

Fork the UISpell repository to your GitHub account and clone it locally using git clone.

2

Create a Branch

Create a new branch for your feature or fix using git checkout -b feature/your-feature-name.

3

Make Your Changes

Implement your changes, following our coding standards and documentation guidelines.

4

Commit and Push

Commit your changes with a clear message and push to your fork using git push origin feature/your-feature-name.

5

Create Pull Request

Submit a pull request from your fork to the main repository, describing your changes and their benefits.
